# Cricket Match Performance Prediction

## Overview
This project predicts cricket runs and wickets using Machine Learning and stores prediction results in MySQL.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
- MySQL

## Features
- Exploratory Data Analysis
- Runs Prediction
- Wickets Prediction
- MySQL Database Integration
- Data Visualization

## ML Algorithm
Random Forest Regressor

## Dataset
300 cricket performance records.

## Workflow
Dataset → EDA → ML Model → Prediction → MySQL Storage

- Power BI Dashboard
