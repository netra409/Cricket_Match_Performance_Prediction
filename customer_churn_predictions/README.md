# Customer Churn Prediction for Subscription-Based Businesses

## Project Overview
This project predicts customer churn using Machine Learning models.
Businesses can identify customers likely to leave and take retention actions.

## Technologies Used
- Python
- Machine Learning
- MySQL
- Power BI

## Machine Learning Models
- Logistic Regression
- Random Forest Classifier

## Features
- Customer churn prediction
- Feature engineering
- Data preprocessing
- Model comparison
- CSV export
- MySQL integration
- Power BI Dashboard

## Dataset Features
- Age
- Gender
- Subscription Type
- Monthly Charges
- Tenure Months
- Total Spend
- Last Login
- Churn

## Project Workflow
1. Load dataset
2. Clean and preprocess data
3. Feature engineering
4. Train ML models
5. Evaluate models
6. Export predictions
7. Store results in MySQL
8. Visualize in Power BI

## Power BI Dashboard
Dashboard includes:
- KPI Cards
- Churn Rate Analysis
- Subscription Insights
- Feature Importance
- High Risk Customers
